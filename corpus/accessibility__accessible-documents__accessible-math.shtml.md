# Creation of Accessible Mathematic Equations

## MathML

MathML displays mathematical and scientific content on the web. It is an application of XML that is supported by many browsers needing only a text editor, though this program has many tools available to help.  This program was developed by the World Wide Web Consortium, an international community that works to develop Web standards.  Guides, tools, and information can be found at [MathML Homepage](https://www.w3.org/Math/).

## LaTeX

LaTeX is a document preparation system for high-quality typesetting with features that let the user make advanced typesetting of mathematics with AMS-LaTeX and typesetting of complex mathematical formulas. Guides, tools, and information can be found at [LaTeX Homepage](https://latex-project.org/).

## Resources

## Tools:

* [jsMath](https://www.math.union.edu/~dpvc/jsMath/)is a system for embedding mathematics in web pages that uses TeX form.

* [The Rehabilitation Engineering Research Center for Wireless Technologies (RERC)](https://wirelessrercarchive.gatech.edu/content/projects/ideal-group-reader.html) provides information and a link about IDEAL Group Reader, an accessible eBook reader supported by MathML.

* IDEAL Group Reader Mathwriting recognition feature is an addition to the [RERC's IDEAL Group Reader app](https://wirelessrercarchive.gatech.edu/node/205.html) for Android and converts handwritten math equations into LaTeX and MathML.

### How-To’s:

* The [CSUN Universal Design Center webpage](https://www.csun.edu/universal-design-center/math-accessibility) on math accessibility gives tips, information, examples, and resources about MathML and MathJax.

* The [National Center for Accessible Media (NCAM)](http://ncamftp.wgbh.org/ncam-old-site/invent_build/web_multimedia/accessible-digital-media-guide/guideline-h-multimedia.html) created a document called Accessible Digital Media Guidelines that features a section to “provide access to scientific and mathematical expressions for all users with disabilities.”.

* [How can publishers create accessible math textbooks?](https://www.washington.edu/doit/how-can-publishers-create-accessible-math-textbooks) is a page provided by UW’s DO-IT program and describes how the user can use MathML and LaTeX to create accessible math textbooks.

* [What are some techniques for creating Braille math materials?](https://www.washington.edu/doit/what-are-some-techniques-creating-braille-math-materials) is a page that describes how Braille can be used in math and how MathML and LaTeX play an important part in this process, courtesy of UW’s DO-IT program.

### Related Articles:

* [STEM Access for the Blind and Visually Impaired (PDF)](http://stanford.edu/~kartiks2/stem-access.pdf) provides an in-depth look at the problems still faced when including blind or visually impaired students in STEM written by Kartik Sawhney, a Stanford student.

* [NCAM’s resources for describing STEM images, DTB Standards, and National Instructional Materials Access Center (NIMAS)](http://ncamftp.wgbh.org/ncam-old-site/experience_learn/educational_media/stemdx/guidelines.html) links.
