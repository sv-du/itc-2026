# Font Awesome Icon Tutorial

## Instructions

In Cascade, there are specific modules where you can use icons, they will be marked by **ICON USABLE** on template instruction pages.

![iconusable](../icon_images/iconusable1.png)

You can use the search feature in the table below for the different icons that are authorized by Cal Poly Pomona for Cascade use. Please do not use icons other than these approved ones. If you wish to request a new icon, email brand@cpp.edu

Once you have decided on an icon, copy the code section and paste it in the icons area.

![iconselect](../icon_images/iconoption.png)

**Example: Clipboard Icon**

1. Find icon on table, copy code from that column.

![selecticon](../icon_images/iconselect.png)

2. Paste it into the Font Awesome Icon box, then save your changes.

![iconpaste](../icon_images/iconpaste.png)

3. It will appear like below.

![clipboard icon](../icon_images/clipboardicon.png)

**Note:** You will not need to worry about color contrast, it should populate as a gold icon on a dark green background.
