# Submitting Content

## Logging In

Submitting Content onto Cascade has two different workflows they are Publish to Test and Publish to Production. This step is for content that is either a file, content page or navigation bar.

[**Click here to learn about Adding Content**](https://www-test.cpp.edu/green-and-gold/tutorial/instructions/getting-started/adding-content.shtml)

## Publishing To Test

### Submit and Review in Test

Once completed editing/adding your content, click on the Preview Draft Button.

![content page preview](img/submit-content/content-page-preview.png)

Select the Submit button. A new popup will appear. On workflow, select Publish to Test. Then, click on Check Content & Submit.

![content page submit test](img/submit-content/content-page-submit-test.png)

A new popup will appear for the review process. In this section, click and review the contents of all three buttons. Once completed, select the finish button that is shown as a checkmark.

![content page review](img/submit-content/content-page-review.png)

### Workflow and Preview in Test

To finish changes, click on the Start Workflow button.

![content page submit test 1](img/submit-content/content-page-submit-test-1.png)

Once completed, click on the go back to page hyperlink.

![content page submit test 2](img/submit-content/content-page-submit-test-2.png)

To view the page, click on the More button. On the new popup, click on the Live button to view the website.

![content page live](img/submit-content/content-page-live.png)

To view the website, make sure to have your **CPP VPN** enabled. Next, on the new page, go to the address bar and in-between 'w' and '.' type in '-test'.

![content page submit test view](img/submit-content/content-page-submit-test-view.png)

For step-by-step instructions on connecting to the **CPP VPN**, refer to this eHelp tutorial:

[eHelp - Virtual Private Network (VPN)](https://cpp.service-now.com/ehelp?id=kb_article&sys_id=c343853247a6229094d59d12736d430d&spa=1)

## Publish To Production

### Submit and Review in Production

Once completed editing/adding your content, click on the Preview Draft Button.

![content page preview](img/submit-content/content-page-preview.png)

Select the Submit button. A new popup will appear. On workflow, select Publish to Production. Then, click on Check Content & Submit.

![content page submit production](img/submit-content/content-page-submit-production.png)

A new popup will appear for the review process. In this section, click and review the contents of all three buttons. Once completed, select the finish button that is shown as a checkmark.

![content page review](img/submit-content/content-page-review.png)

### Workflow and Preview in Production

To finish changes, click on the Assign this step to me button.

![content page submit production 1](img/submit-content/content-page-submit-production-1.png)

On that same location, a new option will show click on the Approve hyperlink to continue.

![content page submit production 2](img/submit-content/content-page-submit-production-2.png)

A new window will appear on top. Click on Submit Content to continue.

![content page submit production 3](img/submit-content/content-page-submit-production-3.png)

Once completed, click on the go back to page hyperlink.

![content page submit production 4](img/submit-content/content-page-submit-production-4.png)

To view the page, click on the More button. On the new popup, click on the Live button to view the website.

![content page live](img/submit-content/content-page-live.png)
