# Application Information

## Applying to the Singelyn Graduate School of Business

Each business master’s program has different admissions requirements and cost of attendance, so please refer to the master’s program page for which you are applying before submitting your application.

When you’re ready to apply, you will be submitting your application through [Cal State Apply](https://www.calstate.edu/apply#Cal%20State Application "Cal State Application")

## Important!

Applications for the MBA, General Accountancy, and MSIS Online are listed under **Cal Poly Pomona Graduate** in [Cal State Apply](https://www.calstate.edu/apply#Cal%20State Application "Cal State Application")

Applications for the MSBA, MSDM, MSDSCM, MSFA, MSHRL, MSIS Face-to-Face, and Accelerated MSA programs are listed under **Cal Poly Pomona Extension** in [Cal State Apply](https://www.calstate.edu/apply#Cal%20State Application "Cal State Application").

## Cal State Apply:

You will need to create a profile and submit all materials through Cal State Apply. There is no cost to create a profile with Cal State Apply, which allows users to upload and organize application materials. A non-refundable fee of $70 is required before the application can be submitted.

If you have any questions regarding application requirements, please contact the [SGSB Office](contact.shtml). If you have any questions regarding Cal State Apply, please contact the main [Admissions Office](https://www.cpp.edu/admissions/index.shtml#Admission%20Office "Admission Office").

## Transcript Information

Student should send their final official transcripts electronically. Applicants only need to provide bachelor's degree transcripts. If your institution does not offer electronic transcripts, we recommend the following services:

· National Student Clearinghouse

· Credential Solutions

If sending PDF transcripts, please indicate the recipient email is verify@cpp.edu. If sending physical copies, please mail to: Cal Poly Pomona Attn: Admissions & Enrollment Planning 3801 West Temple Avenue Pomona, CA 91768
