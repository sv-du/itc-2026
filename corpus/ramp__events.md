# RAMP Events

## Mark your calendar!

### RSVP for events on myBAR

Events will be held in-person or via Zoom. Be sure to check the location when registering for an event (online meeting links are displayed on myBAR for each event). Events may be added throughout the semester; be sure to check RAMP's myBAR page periodically.
Please contact ramp@cpp.edu if you have any questions.
