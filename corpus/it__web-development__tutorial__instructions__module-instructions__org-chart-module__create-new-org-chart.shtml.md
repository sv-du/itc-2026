# Adding Brand New Organizational Chart

## Organizational Chart Instructions

### Adding the Module to Your Site

### **Step 1**

Search for the **“marketplace”** site in Cascade using the top search bar, then select the **Org Chart**module from the left-hand menu.

![](img/step1.png)

### **Step 2**

Click on the more menu and choose copy.

![](img/step2.png)

### **Step 3**

Choose the site and folder you want to copy the module to.

![](img/step3.png)

### **Step 4**

Start workflow.

![](img/step4.png)

### **Step 5**

Complete workflow.

![](img/step5.png)

### **Finished**

You can now edit the module in your own site.

![](../img/img-organizational-chart/step6.png)

### Organizational Chart Module Tutorial

For this section, you'll need:

* Role Title and Role Holder Name for Top Level Position Title and Name
* Role Title and Role Holder Name for Direct Report Position Title and Name (Optional)
* Role Title and Role Holder Name for Secondary Level, Third Level, Fourth Level, and subsequent sections
* Link to an Existing Organizational Chart (Optional)
