# Search/View/Print Reports

![Department Reports](../../img/college_report.jpg)

## Department Reports

### Department Reports

×

To filter the department reports you can go to the following menu:![Appointment filter](img/appointment-filter.png)

You can edit the Academic Year by selecting the dropdown in the Academic Year section:

![Academic year field](img/academic-year-field.png)

You can edit the Form Type and select the dropdown in the Form Type field section:

![Form type field](img/form-type-field.png)

You can edit the Department IDs select the options in Dept ID section:

![Dept ID Field](img/dept-id-field.png)

To select multiple Department IDs hit ctrl + left click on Windows or CMD + left click on Mac.

![Dept ID Field Multiselect](img/dept-id-field-multiselect.png)

Once all the filter options are selected click the ![filter button](img/filter-button.png) button.
To remove all filter options click the ![clear filter button](img/rename-clear-filter.png) button.

To sort the filter results select the header titles:
(Ex. To filter Faculty Name alphabetically select Faculty Name)

![Dept Records Appointments](img/dept-records-appointments.png)

To export the filtered results click the ![Export Report button](img/export-report-button.png) button.

Close

## Faculty reports

###

×

To search the faculty reports, first find the faculty member's Bronco ID you want to search up.

Go to the View/Edit/Print Form View and in the Search for an appointmnet panel search for the Faculty's last name in the corresponding panel:

![View/Edit/Print Form Views](img/view-edit-print-form-view.png)

Under Search Appointments search for the last name of the faculty member:

![Faculty View Lookup](img/faculty-view-lookup.png)

Click the ![Search Button](img/search-button.jpg) button and find the under the results click the Form Number associated with the faculty member:

![Faculty Lookup Result](img/faculty-view-lookup-result.png)

In the Form Details copy the Bronco ID in the Faculty Bronco ID field:

=![Faculty Lookup Details](img/faculty-view-lookup-details.png)

After copying the Faculty Bronco ID go to the Search/View/Print reports tab:

![View/Edit/Print Form Views](img/search-view-edit-reports-view.png)

Click on the Faculty Report Card:

![Faculty report Card](img/faculty-report-card.png)

Under Search for person paste the Bronco ID into the Bronco ID field and click on the blue Bronco ID field to get to the faculty member's page:

![Faculty View Search](img/faculty-view-search.png)

You can view the general report for faculty in the Faculty/Staff Report:

![Faculty View Report](img/faculty-view-report.png)

You can view the faculty member's assigned hours in the Academic Year Hours Report:

![Faculty View Hours](img/faculty-view-academic-hours.png)

You can view the faculty member's appointments on the bottom section:

![Faculty View Appointment Info](img/faculty-view-appointment-info.png)

Close
