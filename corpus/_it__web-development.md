# Web Development

## Welcome to Web Development

IT Web Development supports the production of high-quality web pages. We emphasize all aspects of quality:

* Useful and accurate content
* Understandable navigation
* Attractive layout, both visually and for alternative devices such as screen readers using accessibility standards and best practices including WCAG 2.0AA and WAI-ARAI 1.0
* Correct use of HTML, CSS, and Javascript
* Effective integration with the rest of the Cal Poly Pomona content management system

The services of IT Web Development are aimed at institutional web pages such as divisions, colleges, departments, institutes, centers, and programs. Faculty interested in developing instructional web pages or publishing a personal CPP web page should instead contact [CAFE](https://www.cpp.edu/cafe/) or [eHelp](https://cpp.service-now.com/ehelp?id=sc_cat_item&sys_id=a57b4cf4dbbe0d10d9acca32399619ef&sysparm_category=d6f972c313ab130027a43ff18144b0a4 "eHelp website").

Student clubs and organizations can request webspace on sites.cpp.edu with a single page or multipage template. These sites are self-supported and can be requested through the [Service Desk](https://cpp.service-now.com/ehelp?id=sc_cat_item&sys_id=fa1ad918db295c10e00f785e0f961982&sysparm_category=d571b2f26f84820025ce9e4eae3ee482).
