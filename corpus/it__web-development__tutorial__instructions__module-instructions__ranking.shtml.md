# Rankings Module Tutorial

## Instructions

In this Module Rankings for the College of the Department are required, at least one ranking is required for the page and the max number of rankings that can be added are six.

**Rankings must be sourced exclusively from [cpp.edu/aboutcpp/rankings](https://www.cpp.edu/aboutcpp/rankings.shtml). If a specific ranking is not available, select the most relevant general Cal Poly Pomona ranking listed on that page.**

***Page View***![](../img/ranking-module-site-example.png)For this section you will need:

* 3 rankings
* Titles and sources for the rankings; Note: If you don’t have rankings that apply to your college, use University Rankings and Statistics.
* 3 statistics about your college
