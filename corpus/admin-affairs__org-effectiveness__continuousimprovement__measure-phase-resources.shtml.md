# Measure Phase Resources

## ![](../pictures/measure_small.png)Measure Phase: *How does the process currently perform?*

These tools and templates will help you map your current process and establish meaningful baseline data against which you will be able to measure data from your future process.

This table provides resources for the Measure Phase of a continuous improvement project.

|  |  |  |  |
| --- | --- | --- | --- |
|  | **Template/Tool** | **User(s)** | **Description** |
| **High Level Process Mapping** | [Process Mapping Template (xlsx)](processmappingtemplate.xlsx) | Facilitators | Process Flowcharts visually display the steps in a process in order to identify waste. This tool can also be used to demonstrate the change that has taken place between the “Current State” and the “Future State” of the process. |
| [SIPOC (xlsx)](sipoctemplate.xlsx) | Facilitators | SIPOC can be used for high level process mapping. |
| **Baselining** | [Attribute Sigma Calculator (xlsx)](attributesigmacalculator.xlsx) | Facilitators | Baselining the current state performance is a key step in the Measure phase.  Use this Attribute Sigma Calculator to calculate the Sigma level of the process.  Consult with your Black Belt if you are working with continuous data. |
| **Other Tools** | [Project Benefit Calculator (MS Excel)](processimprovementvaluecalculator.xlsx) | Facilitators & Managers | For the Measure phase, use this calculator to estimate the benefits of the project.   * Converting work hours into monetary value for staff and MPP’s * Compiling the monthly material cost and savings * Use this calculator to finalize the savings in the Improve / Control Phase. |
