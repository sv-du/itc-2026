### Common Data Set

**The Common Data Set (CDS)** initiative is a collaborative effort among data providers in the higher education community and publishers. The combined goal of this collaboration is to improve the quality and accuracy of information provided to all involved in a student's transition into higher education, as well as to reduce the reporting burden on data providers. The CDS is a set of standards and definitions of data items rather than a survey instrument or set of data represented in a database.
