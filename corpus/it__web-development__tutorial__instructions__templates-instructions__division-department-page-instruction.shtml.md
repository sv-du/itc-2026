# Division Department Landing Page Instruction

## Introduction

To start creating content for your website, go to the top left corner next to the site menu, and click on the Add Content button. Next, click on the Entity folder button and choose the Department Landing Page template.

![adding content](../img/col-department-landing-page-addcontent.png)

This checklist breaks down each element of the new CPP web template for the [Department Landing Page](https://www-test.cpp.edu/green-and-gold/mockups/division-landing-page.shtml). Use this as an interactive guide to collect all the content you will need for your page and you will be set up for success during the content entry process in Cascade. If you are using this template and are not a college, you will need to adjust your page accordingly and may find some guidelines are not applicable to you. It may be helpful to have both the template, and this checklist open simultaneously. If you require assistance understanding what template to use, please submit a   [ticket](https://cpp.service-now.com/ehelp?id=sc_cat_item&sys_id=80cc3358db951c50e00f785e0f961996&sysparm_category=d571b2f26f84820025ce9e4eae3ee482).

**Please note: all elements are required and should not be removed unless otherwise noted in bold. All character counts include spaces.**
