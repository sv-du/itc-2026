# Faculty and Staff Resources

## Request Forms

[### Data and Communication Request

Submit a request for student data (query/report) or submit a request to send communications to applicants/admitted students.](https://forms.office.com/r/KrG0xzA0bm "opens a new window")

[### Tableau Dashboard Access Form

Most Tableau dashboard permissions are based on an employee’s role. However, some dashboards require additional approvals for access. This form should be completed by a HEERA manager (MPP) or by a HEERA manager delegate.](https://cpp.service-now.com/ehelp?id=sc_cat_item&sys_id=a4237f972bb426d0d0ebf5dfc391bfa1&sysparm_category=b298635b6f1161000a149e4eae3ee405&catalog_id=-1 "opens a new window")

[### PeopleSoft CS Access Request

Most PeopleSoft Campus Solutions (Student Information System) permissions are based on an employee’s role. However, some functionality require additional approvals for access. This form should be completed by a HEERA manager (MPP) or by a HEERA manager delegate.](https://cpp.service-now.com/ehelp?id=sc_cat_item&sys_id=05ba1beedb4c0450e00f785e0f9619c3&sysparm_category=b298635b6f1161000a149e4eae3ee405 "opens a new window")

[### Dashboard Enhancement Request

Submit a request form to edit any of the Class Schedule dashboards on Tableau. For access to specific dashboards, or requests to edit any dashboards not in the Class Schedule dashboard tab, please refer to the "Tableau Dashboard Access Form".](https://forms.office.com/pages/responsepage.aspx?id=HqZLFuw5XU-J_6ofAKUhtALfAHtQJlFIq-EkKDpEQDFUOUVOQklDUklWNkxNRjhVT1ZRV0ZZMDVDOCQlQCN0PWcu&route=shorturl "opens a new window")
