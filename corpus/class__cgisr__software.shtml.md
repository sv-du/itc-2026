# Software

## Information

The CGISR recommends ArcGIS Pro 3.5. At this time. Any current CPP account holder may download and install Pro onto their computer for education and research purposes. The software is also available virtually and on some IT&IP managed computers across campus.

## Availability

Virtual access to ArcGIS Pro can be found in the [Virtual Software Lab](https://www.cpp.edu/it/virtual-software-lab/index.shtml) (under the 'Graphics Lab' option). On campus, you can find the app in the 24-hour lab, library computers on floors 4 & 5, and the CGISR (5-215).

## Downloads

### ArcGIS Pro 3.5.5

ArcGIS Pro 3.5 - [Download](https://livecsupomona.sharepoint.com/%3Au%3A/s/itip_published/ERk8GS0zg_FNsMIWTX18ldgB_MVTbbL7Q5SA6SuJeL7-SQ?e=AegdyR) (Patch 5 only - [Download](https://livecsupomona.sharepoint.com/%3Au%3A/s/itip_published/IQB78IvNxveeTok3hzVYa9hkAe2FDyVB_mJjfjtLxmuSSxw?e=AWy3aY)) - Patch details on bugs fixed can be found [here](https://pro.arcgis.com/en/pro-app/3.5/get-started/release-notes.htm)

* Installation Instructions – [Download (PDF)](docs/35_proinstall.pdf)
* Licensing Instructions – You must have a CPP ArcGIS online organization account to use ArcGIS Pro. Please use your CPP account to create one [here](https://www.cpp.edu/class/cgisr/resources/arcgis-online-steps.shtml "Cal Poly Pomona ArcGIS Online Login").
* Microsoft .NET 8.0.x Desktop Runtime is required to run ArcGIS Pro 3.5. Download and install this package if you experience this issue.
