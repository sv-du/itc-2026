# Parking Permit Pay Stations

#### Instructions for Permit Dispensers

When purchasing a permit at a Dispenser, press any key to activate machine. Dispenser will accept credit/debit cards, currency up to $8.00 and coins.

#### Parking Permit Display & Purchase

Permits purchased from the parking permit pay stations are valid in student lots only and must be displayed face up on the driver’s side dashboard of the vehicle with the expiration date and time fully visible. Do not obstruct the Vehicle Identification Number (VIN). Permits are not valid in Faculty/Staff, Reserved, Rideshare, meters, Service, and time controlled areas (30 minute zones).

Information regarding purchasing a quarterly permit can be obtained from [Parking and Transportation Services Website](https://csupomona.t2hosted.com/cmn/auth_ext.aspx).

#### Daily and Weekend Permit Price

A daily permit Monday-Friday is $8.00 each day.  A daily permit Saturday-Sunday is $5.00 each day.

* Parking Permit required at all times
* Location of Permit Dispensers
* [Map of the permit dispensers on campus](https://www.cpp.edu/maps?id=1130#!ce/25102?ct/25111,25103,0,29218?mc/34.05655044903938,-117.82282186994755?z/16?lvl/0)

#### Represented Employee Daily Parking Rate

Represented CPP employees may [request an Employee Daily Parking Rate Code](https://app.smartsheet.com/b/form/36fe913414624e458da3923c94086672) to be used at any parking permit machine (excluding Lot C & G). Once the code is entered the $8 daily rate will be reduced to a $3 daily rate. All machines accept credit card payment, limited machines accept cash. This parking permit is valid in any Faculty/Staff parking space. The code will expire on the last day of the month the request was made.

* Must show as 'active CPP employee' in university system
* Must be represented CPP employee
* Must not have an annual or semester parking permit on file
* Student Assistants, Teaching Associates and Grad Assistants are not eligible
* Code is limited to one use per day
* Code is nontransferable
* Code may be request each semester
* Allow a minimum of 2 business days for processing
* Deviation from program rules may impact eligibility
