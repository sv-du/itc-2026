# Course Information Module Tutorial

## Instructions

This module comes with a **Courses** and an **Advising** button. You can add additional buttons if necessary.

***Page View***

![](img/green-and-gold-tutorial-img/program-course-information-page.png)

For this section you will need:

* 2 sentence paragraph describing the college program's courses
* 2 buttons (one for courses, another for advising), additional buttons are optional
* Links for each button
  + Course button should link to the course catalog
  + Advising button should link to the college program's advising page

***Cascade View***
