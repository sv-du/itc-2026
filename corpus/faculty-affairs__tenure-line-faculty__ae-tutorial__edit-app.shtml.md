## Print Form

### Print Form

×

1. **Click on Print Form** to open a new browser tab containing the appointment details in a print-friendly format.  ![print pdf form](img/print-pdf-form.jpg)

2. The form will open in a new page. The layout may vary depending on the appointment or form type, but it will look similar to the example shown.

![Print page](img/print.jpg)

3. To print or save the file, open the browser’s print menu by selecting the  **three-line (“hamburger”) menu**  in the top-right corner of the browser, or by pressing  **Ctrl + P**  on your keyboard. This will open the Print dialog box, similar to the example below.

![print dialog](img/print-dialog.jpg)

4. In the Print dialog, set **Destination** to **“Print to PDF.”**
 Then choose a location on your computer where you would like to save the PDF version of the appointment.

**Note:** Printing or saving to PDF using the steps above will include **both the appointment form and the memorandum** in the final document.

Close

## Edit Form Hours

### Edit Form Hours

×

The Working Dates screen will be activated once you click on working dates tab:

![working dates tab](img/working-dates-tab.png)

The first section will show the working dates entered for the current appointment like so:

![working dates appointment](img/working-dates-appointment.png)

You can see each working date's assigned semster, date, the number of hours worked on that date and the total number of hours from all working dates in the appointment.

To remove a working date for an appointment select the![add button](img/delete-button.png)button.

To insert a new working date go to the following Insert New Working Date Section: ![Insert new working dates](img/insert-new-working-dates.png)

| **Field** | **Explanation** |
| --- | --- |
| ![select semester](img/select-semester.png) | You can select the desired Semester in this section. |
| ![select year](img/select-year.png) | You can select the desired year for the Calendar view in this section. |
| ![select month](img/select-month.png) | You can select the desired month for the Calendar view in this section. |
| ![edit date hours](img/edit-date-hours.png) | You can edit/add form hours by inputting the number of hours worked on a date in this section. |

To submit the changes select the ![add button](img/add-button.png) button. This will update the Working Dates for the appointment.
