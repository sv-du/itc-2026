## More Information

### How to Apply for the Cal Vet Fee Waiver

1. Obtain following documents:
   * Birth Certificate
   * Service-Connected Disability document for Veteran
   * Application
     + If student did not file for taxes, a non-filing statement from the IRS will be required.
     + If student did file taxes, a tax transcript or income taxes will need to be submitted
2. Submit all documents to a VA County Office
   * VA County Office can be located by clicking this link: [CalVet Service Provider Search](https://www.calvet.ca.gov/find-a-service-provider).
     + Type in your Zip Code
     + Enter Maximum distance to search
     + Check the box next to “Education”
     + Click on the bolded County *(i.e Los Angeles County, Riverside County, Orange County, etc.)*

### What the Cal Vet Fee Waiver Covers

The Cal Vet Fee Waiver covers tuition only. It does not cover mandatory auxiliary fees. The Cal Vet Fee Waiver can cover the $55.00 application fee, but needs to be requested. Please make the request to the Veterans Resource Center.

### More About the Cal Vet Fee Waiver

* The Cal Vet Free Waiver needs to be **renewed** annually.
* The Cal Vet Fee Waiver covers Fall Semester to Summer Semester.
* It is highly recommended that the student brings the Cal Vet Fee Waiver directly to the VA Certifying Official or to the Veterans Resource Center.
* Complete the Cal Vet Fee Waiver application process as soon as possible in the Summer or possibly Spring to have it ready for Fall.
* Students should **check** their **e-mail on a daily basis**. All communication and information is provided to students CPP e-mail.
* Never Assume. **If you have a question or are unsure of an answer, please do not hesitate to ask.**
