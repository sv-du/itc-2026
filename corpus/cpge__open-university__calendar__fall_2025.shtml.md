# Fall 2025

## Term Calendar

| **Fall 2025** | |
| --- | --- |
| **Term Calendar** | |
| August 11 @ 9am - September 4 @ 4pm | Open University petitions accepted |
| August 21 | Classes Begin |
| December 8 - 14 | Final Exams |

## Installment Plan Due Dates

| **Installment Plan Due Dates** | |
| --- | --- |
| Installment | **Due Date** |
| Payment #1 | Due after petition is approved |
| Payment #2 | October 10, 2025 |
| Payment #3 | November 14, 2025 |

## Refund Timeline

| **Refund Timeline** | |
| --- | --- |
| **Date** | **Refund Amount** |
| Before August 21 | 100% Refund |
| August 21 - 26 | 90% Refund |
| August 27 - September 2 | 80% Refund |
| September 3 - 9 | 70% Refund |
| September 10 - End of the Term | No Refund Given |
