# Policy Prohibiting Harassment

|  |  |
| --- | --- |
| Subject: | Policy Prohibiting Harassment |
| Policy Number: | 1.3 |
| Date Adopted: |  |
| Last Revised: |  |

## Policy Statement:

Cal Poly Pomona adheres to the California State University (CSU) system policies embodied in the CSU executive orders (e.g. EO 926, 1088, 1095, 1096, 1097 and 1098) reflecting these laws.

## Implementation:

[Office of Institutional Equity & Compliance](https://www.cpp.edu/officeofequity/)
