# News Feed Instructions

## Introduction

The news module is not available by default. If a college or department want a news module on their website, please submit a ticket  requesting the news module.

[submit a ticket](https://cpp.service-now.com/ehelp?id=sc_cat_item&sys_id=0f7a475ddbdd9010e00f785e0f96198c&sysparm_category=d571b2f26f84820025ce9e4eae3ee482)

**The module is available for use across the following template pages:**

* [College Department Page](../templates-instructions/college-department-page-instruction.shtml)
* [Division Department Page](../templates-instructions/division-department-page-instruction.shtml)
* [Content Subpage](../templates-instructions/content-subpage-instruction.shtml)
* [College Landing Page](../templates-instructions/college-landing-page-instruction.shtml)
* [Division Landing Page](../templates-instructions/division-landing-page-instruction.shtml)

**There are two types of news modules:**

1. PolyCentric News Feed: Displays news pulled directly from PolyCentric .
2. A custom news feed: A department-managed news feed built internally for a specific website.
    If you would like to learn how to create and set up a custom News Module, visit the [News Module Instructions](news-module.shtml) page.

**Important Notes**

* This tutorial covers how to select and apply a pre-configured news feed to a page.
* A single site can use multiple news feeds, depending on which feed is selected on each page.
