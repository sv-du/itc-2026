# Improve Phase Resources

## ![](../pictures/improve_small.png)

## Improve Phase: *How will the team mitigate the root causes of the problem?*

 You can use these tools and templates to identify potential solutions, map out and plan your new process, and identify risk factors.

This table provides resources for the Improve Phase of a continuous improvement project.

|  |  |  |
| --- | --- | --- |
| **Template/Tool** | **User(s)** | **Description** |
| [Countermeasure Matrix Template (xlsx)](countermeasurematrixtemplate.xlsx) | Facilitator & Project Team | The Countermeasure matrix should be populated with root causes from the Analyze phase.  In Improve, the team will develop solutions and countermeasures to address the root causes. |
| [Failure Modes Effects Analysis (xlsx)](fmeatemplate.xlsx) | Facilitator & Project Team | The Failure Modes Effect Analysis tool or  FMEA for short, is used to analyze any potential failures in the future state process. |
| [Value Calculator & NPV](1benefitvaluenpvcalculator.xls) | Project Leader / Green Belt & Team | This template is used to capture the annual savings / benefits resulting from the process improvement project.  It also provides an NPV calculation of the benefits over 5 years. |
