## Image Optimization Tools

[### Resizing an Image Using Photos

Quickly adjust image dimensions in Photoshop to optimize for faster loading times without sacrificing quality.](#modal-0 "Resizing an Image in Photoshop")

### Resizing an Image in Photoshop

×

1. **Open the Image in Photoshop:**
   1. Launch Photoshop and open the image file you want to resize.
2. **Access the Image Size Menu:**
   1. Navigate to Image > Image Size in the top menu.
3. **Adjust the Dimensions:**
   1. Check the box next to the chain icon to preserve aspect ratio.
   2. Reduce the width (e.g., to 2000 pixels for web use). The height will automatically adjust if the chain icon is active.
4. **Check Image Quality:**
   1. Zoom in using Ctrl + Plus (Windows) or Command + Plus (Mac) to ensure the resized image maintains acceptable quality.
5. **Export the Resized Image:**
   1. Go to File > Export > Export As.
   2. Choose the desired format:
      1. JPEG: Best for photographs, smaller file size.
      2. PNG: Use if transparency is required, though it generally results in larger files.
   3. Adjust quality settings as needed to further reduce file size without losing visual quality.
6. **Save the File:**
   1. Select a location to save the resized image and click Export.

**Notes:**

* Aim for file sizes in kilobytes (e.g., 50KB–300KB for web images). Avoid large megabyte-sized files for better website performance.
* Web images typically require 72–150 DPI. Avoid unnecessary high resolutions meant for print.
* Always maintain the original aspect ratio to avoid stretched or distorted images.
* JPEG files are efficient for photographs due to smaller file sizes with good quality. Reserve PNG for graphics with transparency.
* Optimizing image sizes ensures faster page load times and better user experience.
* Always keep a copy of the original image in case you need to make adjustments later.
* Test the resized image on various devices (e.g., desktop, tablet, mobile) to ensure it looks good across platforms.

Close

[### Cropping an Image in Photoshop

Focus on what matters by cropping out unnecessary elements to enhance composition and reduce file size.](#modal-1 "Cropping an Image in Photoshop")

### Cropping an Image in Photoshop

×

1. **Open the Image File:**
   1. Launch Photoshop.
   2. Navigate to File > Open and select the image(s) you want to crop.
2. **Check Image Dimensions (Optional):**
   1. Go to Image > Image Size to view the current dimensions.
   2. If necessary, adjust the size by entering the desired width and height while maintaining the aspect ratio.
3. **Select the Crop Tool:**
   1. On the left toolbar, locate and select the Crop Tool (icon looks like two intersecting right angles).
   2. Alternatively, press C on your keyboard.
4. **Set Crop Dimensions:**
   1. In the top options bar, set the desired crop aspect ratio by selecting W x H x Resolution.
   2. Enter the specific width, height (e.g., 500 by 334), and resolution.
5. **Adjust the Crop Area:**
   1. A highlighted border will appear around the image.
   2. Drag the corners or edges of the crop area to adjust the size.
   3. Click and drag within the crop area to reposition the selection.
6. **Crop the Image:**
   1. When satisfied, click the checkmark icon in the top bar, or press Enter to confirm the crop.
7. **Zoom and Verify:**
   1. Press Ctrl + (Windows) or Cmd + (Mac) to zoom in and verify the quality of the cropped image.
8. **Save the Cropped Image:**
   1. Navigate to File > Export > Export As.
   2. Choose a file format (e.g., JPEG) and verify the file size.
   3. Save the cropped image to your preferred directory.
9. **Repeat for Additional Images (Optional):**
   1. If cropping multiple images, Photoshop retains the last crop settings.
   2. Repeat the steps to maintain consistent dimensions.
10. **Upload to Your Platform:**
    1. After cropping, you can upload the resized and consistent images to your platform or project.

[### Cropping an Image in Cascade](#modal-2 "Cropping an Image in Cascade")

### Cropping an Image in Cascade

×

Use Cascade’s built-in tools to crop images directly in the CMS—no external software needed.

**1. Access the Image Folder:**
Navigate to the image folder where the image is stored.
**2. Edit the Image:**

Select the image you want to crop and click on the "Edit" option.

![](img/io-step2.png)
**3. Adjust Width and Height:**

1. In the edit window, locate the width and height properties.
2. If the image dimensions don't match your desired aspect ratio, manually enter a new width and height. For instance, you might enter 600 for width and 400 for height.

![](img/io-step3.png)
**4. Enable Cropping:**

If the aspect ratio doesn't automatically align, use the crop tool to adjust the dimensions manually.

![](img/step4.png)
**5. Draw the Crop Box:**

Drag the crop box to frame the portion of the image you want to retain.
Adjust the crop box to match the required dimensions (e.g., 600x400). Note that in Cascade, the crop box proportions aren't constrained automatically, so you must manually adjust to the desired size.

**6. Apply the Crop:**

Once satisfied with the selection, click the crop button to apply changes.

![](img/io-step6.png)

**7. Preview the Cropped Image:**

Use the "Preview" option to check the cropped image for accuracy.

![](img/io-step7.png)

**8. Save and Submit:**

Save the cropped image.
Submit the image through the Cascade workflow for further use or publication.

![](img/io-step8.png)

**Notes:**
Cropping large image files in Cascade can be cumbersome due to performance limitations. It's often better to crop such files in software like Photoshop before uploading.
Cascade's cropping tool doesn't provide detailed control over compression or quality settings, so using dedicated image-editing software like Photoshop is preferred for high-quality results.

## 25Live Event Images

To ensure your event is displayed properly in 25Live, please follow the image format and accessibility guidelines below.

#### **Regular Events**

25Live supports two image formats for regular events:

* **Square image:** Maximum size of **200×200 px**
* **Rectangular image:** **16:9 ratio** (recommended size: **200×112 px**)

All images must be uploaded to **Cascade** within the appropriate site. Once uploaded, include the **image URL** when filling out the 25Live event form.

If the event requester is **not a Cascade publisher**, they must coordinate with their department’s Cascade publisher to upload the image.

#### **Posters**

Poster-sized images are **not accessible** and **not supported** in 25Live.
 Instead:

* Upload a **thumbnail image** using the same specifications as for regular events.
* Include the **event webpage URL** (where the full poster and additional details are available) in the **Event Description** field.

## Image Size Recommendations

Please note that All file sizes should be under **300KB** after saving. The smaller the file size, the faster the load time. Do not drop below the Minimum Size (Width x Height) even if it makes the file size smaller.

Image recommended sizes

| **Section** | **Minimum Size** | **Recommended Size** | **Approximate Aspect Ratio** |
| --- | --- | --- | --- |
| Title | Width x Height | Width x Height | Width : Height |
| Small Hero | 300 × 150 px | 496 × 372 px | 4:3 |
| Large Hero | 580 × 464 px | 1160 × 928 px | 5:4 |
| Hero Slideshow | 500 × 300 px | 1000 × 600 px | 5:3 |
| Picture and Text Carousel | 562 × 342 px | 1124 × 684 px | 3:2 |
| Broncos in Action | 460 × 285 px | 920 × 570 px | 16:10 |
| Broncos in Action Logos | 200 × 155 px | 400 × 310 px | 4:3 |
| News Large | 460 × 285 px | 920 × 570 px | 16:10 |
| News Small | 207 × 160 px | 414 × 320 px | 4:3 |
| Full Width Featurettes | 700 × 500 px | 1400 × 1000 px | 7:5 |
| Circle Featurettes | 460 × 460 px | 920 × 920 px | 1:1 |
| Cards with Pictures and Text | 460 × 351 px | 920 × 702 px | 4:3 |
| Cards with Pictures and Text (Green) | 700 × 377 px | 1400 × 754 px | 16:9 |
