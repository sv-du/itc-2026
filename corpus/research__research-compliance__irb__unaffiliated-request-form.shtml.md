# Unaffiliated Researcher Access Request Form

**Unaffiliated Investigator Access Request**

Email the below information to irb-office@cpp.edu and irb@cpp.edu (a staff member will be confirming receipt of the below information within 48-72 hrs).

**About the Researcher**

1. External Researcher's Name

2. Email

3. Name of Home University, Institution or Agency

4. Email Address of Home IRB

5. Phone Number of Home IRB

6. Project/Research Title

7. Request Type

Conduct research with CPP subjects

Conduct research with CPP existing data

\*Please review the Enrollment Management &Services Request Form for access to internal data

<https://www.cpp.edu/enrollment/resources/faculty-staff.shtml>

8. A brief description of your research & justification for recruiting subjects from CPP, or using CPP existing data

9. How will this research benefit CPP subjects?

10. Upload Approved Consent Forms

    Upload file

11. Upload 'Surveys'

    Upload file

12. Upload 'Interview Guides'

    Upload file

13. Upload 'Recruitment Flyer' or 'Recruitment Emails'

    Upload file

14. Appropriate permission/support letters from the CPP department that you wish to recruit participants from

   Upload file

File types: Word, Excel, PPT, PDF, Image, Video, Audio
