## Frequently Asked Questions

### How do I gain access to Tableau?

All faculty and staff are automatically granted access to the [CPP Tableau dashboard](https://analytics.cpp.edu/) website. Most Tableau dashboard permissions are based on an employee’s role. However, some dashboards require additional approvals for access. HEERA managers, or their delegates, can complete the [Tableau Dashboard Access Request](https://cpp.service-now.com/ehelp?id=sc_cat_item&sys_id=a4237f972bb426d0d0ebf5dfc391bfa1) form should specific dashboard access be needed.

### How do I access Tableau from off-campus?

Tableau and Data Warehouse are available on-campus, or off-campus with a valid VPN (Virtual Private Network) connection. For information on setting up a VPN connection please reference the [eHelp VPN article.](https://cpp.service-now.com/ehelp?id=kb_article&sys_id=c40f854e6fe8e600e020f35d5d3ee4d5)

### If you see an error when logging in

If you see the following error when logging in to Tableau, close all browser windows and then try again.

![Tableau error message: Tableau Unable to Sign In Invalid username or password Try again](../img/tableau-error.jpg)

### Who can I contact for more information?

For more information, please email: irpa@cpp.edu

For a specific contact person, please see our [staff directory](https://www.cpp.edu/data/about.shtml).
